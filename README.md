### Install python libraries
```shell
$ ./node_modules/.bin/bip install <package-name>
```

### Use `Gulp` to install python libraries
- install 
  > `$ gulp 'pip install' --lib jqdatasdk`
  
- sync libraries list
  > `$ gulp 'pip sync'`
